# 研修医マスタ & 専門科ダッシュボード

Excelファイルをアップロードして、研修医データを視覚的に管理・分析するWebアプリケーションです。

## 機能

- **Excelファイルアップロード**: ドラッグ&ドロップまたはクリックでファイルを選択（.xlsx / .xlsm 対応）
- **研修医マスタ一覧**: カード表示・テーブル表示の切り替え、検索・フィルタ・ソート機能
- **専門科ダッシュボード**: 各種グラフによるデータ可視化（横棒グラフ、ドーナツグラフ、折れ線グラフ、積み上げ棒グラフ）

## 技術スタック

- React 18 + TypeScript
- Vite
- Tailwind CSS v4
- SheetJS (xlsx)
- Chart.js + react-chartjs-2
- Lucide React

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# プロダクションビルド
npm run build

# ビルド結果のプレビュー
npm run preview
```

## Excelファイルのフォーマット

以下のカラムを含むExcelファイルを用意してください：

| カラム名 | 説明 |
|---------|------|
| 年度 | 例: 2025 |
| 学年 | PGY1 〜 PGY6 |
| 初・後 | 例: 51期 |
| PHS | PHS番号 |
| 名前 | 氏名 |
| ふりがな | ひらがな表記 |
| 性別 | 男 / 女 |
| 専門科 | 専門科名 |
| 進路 | 進路情報 |
| 動向調査 | 動向調査情報 |
| 本籍 | 出身地 |
| 出身大学 | 大学名 |
| 備考 | メモ |
| email | メールアドレス |

シート名は `main` を推奨します（ない場合は最初のシートを使用）。
